# Mean

### Definition

The arithmetic mean is the sum of all values divided by the number of observations.

It represents the central point of a dataset and it is used to measure central tendency.

### Formula

$$\bar{x} = \frac{\sum_{i=1}^{n} x_i}{n}$$

### Interpretation

- Mean represents the balance point of the data
- It is the value where the dataset would “balance” if placed on a number line

### When to use  it

- When data is numerical and continuous
- When distribution is approximately symmetric (normal-like)
- When no strong outliers are present

### When not to use it

- When data is heavily skewed
- When strong outliers are present

### Example

```
incomes = [3, 3, 4, 50]

sum = 60
n = 4

mean = 60 / 4 = 15
```
