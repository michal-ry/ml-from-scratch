# Maximum

### Definition

The maximum is the largest value in a dataset.

### Formula

$$x_{\max} = \max(x_1, x_2, \ldots, x_n)$$

### Interpretation

- Maximum represents the largest value in a dataset
- It can help identify extremely high values or possible outliers

### When to use it

- When working with numerical data
- To find the largest value
- To identify possible high-end outliers
- To calculate the range

### When not to use it

- When data is categorical
- When you need a measure of central tendency

### Example

```
numbers = [1, 5, 2, 0]

maximum = 5
```
