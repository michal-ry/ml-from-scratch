# Range

### Definition

Range measures the total spread between the minimum and maximum values in a dataset.

### Formula

$$R = x_{\max} - x_{\min}$$

### Interpretation

- Range represents the distance between the maximum and minimum values
- It uses only two values, so it does not describe the full distribution

### When to use it

- When working with numerical data
- To quickly measure the total spread of data
- For fast exploratory analysis

### When not to use it

- When the dataset contains strong outliers
- When you need a robust measure of spread
- When you want to understand how values are distributed inside the range

### Example

```
numbers = [1, 5, 2, 7]

data_range = 7 - 1 = 6
```
